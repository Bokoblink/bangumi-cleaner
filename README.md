## Bangumi 追番纯享 · 模块隐藏脚本

安装地址：[https://greasyfork.org/zh-CN/scripts/542079-bangumi-追番点格子纯享](https://greasyfork.org/zh-CN/scripts/542079-bangumi-追番点格子纯享)

------

为 Bangumi 番剧首页和详情页带来一份**极简纯净的观番体验**，适合只想记录追番进度，不参与讨论、不浏览吐槽或推荐信息的用户。

------

### 功能特色

#### ✅ 首页自动隐藏：

- **右侧边栏**

#### ✅ 番剧详情页自动隐藏：

- **评论**
- **讨论版**
- **吐槽箱**
- **推荐本条目的目录**
- **谁看这部动画**

#### ✅ 章节页自动隐藏：

- **评论**

#### ✅ 人物页自动隐藏：

- **评论**

#### ✅ 极简、安全、轻量

- 不修改页面功能，不上传数据
- 不依赖第三方库，兼容性强
